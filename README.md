## Travel Planner

This app allows users to make their own Travel Planner

### Prerequisites

- Install `node`
- Install `npm`
- Create and fill the .env file with API keys from Geonames, Darksky, and Pixabay like this:

```
GeonamesUsername=********************
darkSkyKEY=**************************
pixabayKEY=**************************
PORT = 8081
```

### How to run

- Clone this repo
- `cd` to the folder
- Run `npm install`

#### Production mode

- `npm run build-prod`
- `npm run start`

#### Development mode

- `npm run build-dev`
- `npm run server`

#### Test mode

- `npm run test`
